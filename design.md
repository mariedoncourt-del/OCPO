# Design Direction — MAF OPCO Assistant

## Typography
- **Display font**: "DM Sans", sans-serif (headings, nav)
- **Body font**: "DM Sans", sans-serif (chat messages, UI text)
- Clean, professional, highly readable

## Color Palette
- **Primary**: #1E3A5F (dark navy blue — trust, authority)
- **Primary Light**: #2A5080
- **Accent**: #3B82F6 (bright blue — interactive elements)
- **Accent Hover**: #2563EB
- **Background**: #F8FAFC (very light gray-blue)
- **Surface**: #FFFFFF (cards, chat bubbles)
- **User Bubble**: #1E3A5F (navy, white text)
- **AI Bubble**: #FFFFFF (white, dark text, subtle border)
- **Text Primary**: #0F172A
- **Text Secondary**: #64748B
- **Border**: #E2E8F0
- **Success**: #10B981
- **Warning**: #F59E0B
- **Danger**: #EF4444

## Layout
- Full-screen chat interface
- Left sidebar with branding + example questions (collapsible on mobile)
- Main chat area centered, max-width 800px
- Input bar fixed at bottom with send button

## Style
- Clean, minimal, professional — financial/consulting feel
- Subtle shadows on cards and bubbles
- Rounded corners (8px bubbles, 12px cards)
- Smooth transitions on hover/focus
- No emoji in UI — use Lucide icons
